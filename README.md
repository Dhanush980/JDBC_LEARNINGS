# JDBC Learnings and Practice Programs

This repository contains my complete JDBC learning journey with hands-on code examples.  
Topics covered include:

- Establishing connection with MySQL/Oracle using JDBC
- Using DriverManager and Connection interfaces
- Executing SQL queries using:
  - Statement
  - PreparedStatement
  - CallableStatement
- Performing CRUD operations (Insert, Update, Delete, Select)
- ResultSet navigation and metadata
- Transaction management (commit, rollback)
- Batch processing
- Exception handling in JDBC
- Basic mini-program examples

This repo is useful for beginners and for interview preparation in Java + SQL.
